# Security Policy

## Educational project (not production cryptography)

This repository is an **educational transparency project**. It is **not** a production-ready cryptography library and is **not designed to protect real data**.

If you need real-world encryption/security, please use well-audited libraries and proven protocols.

## Reporting issues

You are welcome to report:
- crashes, incorrect results, broken build steps
- deviations from known-answer tests / test vectors
- documentation errors or misleading explanations

However, please understand:
- This project does **not** provide security guarantees.
- There is **no bug bounty** program.
- “Vulnerabilities” in the traditional sense are expected in educational code and are usually handled as **documentation/correctness** issues.

## Disclosure

If you believe you found something that could cause harm due to misunderstanding or misuse, please open an issue with clear reproduction steps and a short explanation of the impact.
