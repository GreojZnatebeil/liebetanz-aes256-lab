# 01 Überblick

Dieses Projekt ist ein Lernlabor: Ziel ist, AES-256 und typische Bausteine (Blockchiffre, Runden, Key-Schedule, Betriebsarten) praktisch zu verstehen.

## Was du hier findest
- eine transparente AES-256-Implementierung in Pascal/Lazarus
- Beispiele für Betriebsarten (ECB, CBC) – didaktisch, nicht „produktionstauglich“
- begleitende Erklärungen im Code und in den Docs

## Wichtige Begriffe
- Blockgröße (AES): 128 Bit
- Schlüsselgröße (AES-256): 256 Bit
- Runde / Round: wiederholte Transformationsschritte
- Key Schedule: Erzeugung der Rundenschlüssel
